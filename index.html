Notes!

HTML

  • The head element contains info about the webpage, not visible to user.
  • The body element represents the visible content showm to the user.
  • All Html elements make up the skeleton of webpage.
  • Html elements ARE CASE SENSITIVE :/
  • Semantic elements: clearly describes meaning to browser and developer. (Ex. <"form">, <"table">)

CSS

  • Selector- defines elements to which declarations apply.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • "*" (asterisk)- selector that applies rule to all elements visible on page.
  • external style sheet: allows to add styles to HTML elements while keeping content and style seperate.

Git

  • "git status": checks what branch we are currently on.
  • "git checkout" -b branch-name": creaytes a new branch and switches to it.
  • "git pull": means to update changes from remote repo and merge to main repo.
  • "git commit": saves repo changes to local and not remote repo.
  • "git push": updates commits made and sends to remote repo.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JS IS CASE SENSITIVE
  • Array- groups of related data stored in a single value
  • For loop has 3 important statements:
    1. Starting point, "var x = 0"
    2. condition, "x < shapes.length;"
    3. looping code, "x++"
  • "/*, */"- makes block of code inactive without deleting
